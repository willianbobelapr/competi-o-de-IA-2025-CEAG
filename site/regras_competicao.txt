REGRAS DA PRIMEIRA COMPETIÇÃO DE PROGRAMAÇÃO COM IA CEAG 2025
Colégio Estadual Alto da Glória
Ensino Médio Profissional - Análise de Projeto de Sistemas

1. PARTICIPANTES
- Alunos do 1º, 2º e 3º ano B do Ensino Médio Profissional em Análise de Projeto de Sistemas.

2. ESCOLHA DA LINGUAGEM
- Cada aluno deve escolher UMA das linguagens apresentadas no site (Java, JavaScript, C#, p5.js ou Python) para se especializar durante a competição.
- A escolha deve ser comunicada ao professor.

3. TEMA OBRIGATÓRIO
- O projeto deve obrigatoriamente abordar o tema AGRINHO 2025:
  • "Como a comida do campo chega à cidade"
  • "Alimentação Saudável"
- Os alunos devem explorar, em seus projetos, ideias, soluções ou reflexões relacionadas a esses temas.

4. PUBLICAÇÃO DO PROJETO
- O projeto final deve ser obrigatoriamente publicado em um repositório Git em modo público.
- O link do repositório deve ser enviado ao professor dentro do prazo estabelecido.
- O repositório deve conter todo o código-fonte, arquivos necessários para execução e, se possível, um arquivo README explicando o projeto.

5. PERÍODO DA COMPETIÇÃO
- A Primeira Competição de Programação com IA CEAG 2025 ocorrerá entre outubro e dezembro de 2025.

6. DESAFIOS E PROJETOS
- Serão propostos desafios e/ou projetos práticos para serem resolvidos utilizando a linguagem escolhida.
- Os desafios podem envolver lógica, algoritmos, interfaces gráficas, criatividade e resolução de problemas reais.

7. AVALIAÇÃO
- Os projetos serão avaliados considerando:
  • Originalidade e criatividade
  • Funcionamento e correção
  • Clareza e organização do código
  • Apresentação do projeto
  • Uso adequado dos recursos da linguagem
  • Adequação ao tema AGRINHO 2025
  • Disponibilidade e organização do repositório público

8. APRESENTAÇÃO FINAL
- Cada aluno deverá apresentar seu projeto para a turma e para a banca avaliadora no final do período.
- A apresentação pode incluir demonstração prática, explicação do código e dos desafios enfrentados.

9. PREMIAÇÃO
- Os melhores projetos de cada turma serão premiados com certificados e/ou brindes.
- Haverá menção honrosa para destaques em criatividade, solução técnica, apresentação e abordagem do tema.

10. CONDUTA
- O trabalho deve ser individual, salvo indicação contrária do professor.
- É permitido pesquisar, consultar materiais e pedir ajuda para dúvidas, mas o código final deve ser de autoria do aluno.
- Plágio ou cópia de projetos prontos resultará em desclassificação.

11. DÚVIDAS
- Em caso de dúvidas, procure o professor responsável.

Boa sorte a todos! Escolha sua linguagem, prepare-se e mostre seu talento!
